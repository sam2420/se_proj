# Shopsphere

Shopsphere is a full-featured e-commerce website built using the MERN (MongoDB, Express, React, Node.js) stack. It offers a seamless shopping experience with an intuitive user interface and a powerful admin dashboard for managing products, orders, and users.

---

## 🚀 Tech Stack
- **MongoDB** – NoSQL database to store product, user, and order data  
- **Express.js** – Backend framework for handling API requests and business logic  
- **React.js** – Frontend framework for building a responsive and interactive user interface  
- **Node.js** – Backend runtime environment  

---

## 🌟 Features
### 🛒 **Shopping Experience**
- Browse products by category and search functionality  
- View product details with ratings and reviews  
- Add to cart and manage cart items  

### 💳 **Order Management**
- Checkout with secure payment options  
- Track order status and view order history  

### 👥 **User Accounts**
- Signup, signin, and user profile management  
- Save shipping addresses and payment details  

### 🔐 **Admin Dashboard**
- View and manage products, orders, and users  
- Update product details and stock levels  
- Handle order fulfillment and shipping

## 🛠️ Setup Instructions
1. **Clone the repository**  
```bash
git clone <repo-link>
cd shopsphere
```

2. **Set up the backend**  
```bash
cd backend
npm install
npm start
```

3. **Set up the frontend**  
```bash
cd ../frontend
npm install
npm start
```

4. **Open the app**  
- Frontend will be available at [http://localhost:3000](http://localhost:3000)  
- Backend will be running at [http://localhost:5000](http://localhost:5000) (if not changed)
